Figma

Figma is a fast and powerful designing tool. Space utilizes all major Figma's features such as variables, auto layouts, reusable components and more.

Preview Sapce Figma
The Figma file is NOT a core part of the theme, but at the moment comes as a complimentary. Hence, we don't officially provide support for Figma. The license usage must be followed the terms of the original obtained license from https://themes.getbootstrap.com/product/space-multipurpose-responsive-template

Get started

While this is not part of the theme and we do not promise super-fast updates for the file, we will do our best to keep it up-to-date as soon as possible, and sometimes updates can take longer than expected.

To get started:

  • Download Figma
  • Download Space.fig file from the provided link in README.txt file.
  • After downloading, just open the Space.fig file on Figma. You are ready to rock!
